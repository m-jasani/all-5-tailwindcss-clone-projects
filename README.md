## 🚀 Getting Started

### Prerequisites
- Node.js ≥ 14.x
- npm ≥ 6.x

### Installation & Running

```bash
# Clone the repository
git clone https://github.com/m-jasani/all-5-tailwindcss-clone-projects.git

# Navigate to a project
cd all-5-tailwindcss-clone-projects/niceadmin

# Install dependencies
npm install

# Start development server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## 🔍 Project Breakdown

### 📊 NiceAdmin Clone (`/niceadmin`)

A full-featured **admin dashboard UI clone** of the popular [NiceAdmin](https://bootstrapmade.com/nice-admin-bootstrap-admin-html-template/) template by BootstrapMade — **rebuilt entirely with Tailwind CSS and React** instead of Bootstrap.

The original NiceAdmin is one of the most widely used Bootstrap admin templates. Recreating it in Tailwind CSS required translating every Bootstrap grid, component class, and utility into Tailwind's utility-first equivalents — a strong cross-framework UI challenge.

**What's inside:**

| Section | Description |
|---|---|
| 🧭 Sidebar Navigation | Collapsible sidebar with icon links, active states |
| 📈 Dashboard Page | Analytics cards, stats overview, recent activity |
| 📋 Data Tables | Structured tabular data with clean row/column styling |
| 📊 Charts Section | Visual data representation panels |
| 🔐 Login / Register | Auth form pages with validation UI |
| 👤 Profile Page | User profile layout with avatar and details |
| ⚠️ Error Pages | 404 and error state page layouts |

**Key Technical Highlights:**
- Replicated Bootstrap's 12-column grid logic using Tailwind's `grid` and `flex` utilities
- Built fully responsive layouts using Tailwind breakpoints (`sm:`, `md:`, `lg:`, `xl:`)
- Composed complex sidebar + main content layout using `flex` and `h-screen` patterns
- Implemented interactive UI states (hover, active, focus) with Tailwind's state variants
- Maintained visual parity with the original Bootstrap template — without writing a single Bootstrap class
